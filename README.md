# Building a Game Project - Final Gambit

This project implements **3D rendering** functionality with textures and shaders as well as skeletal mesh animations and more.

Controls:
  ESC - Quit application

Player 1:   
  A - Move Left  
  D - Move Right  
  S - Punch  

Player 2:   
  J - Move Left  
  L - Move Right  
  K - Punch
